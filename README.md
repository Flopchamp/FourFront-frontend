# FourFront Investment Knowledge Club Frontend

>This project is a responsive, interactive frontend assessment built with HTML, CSS, JavaScript, and Bootstrap. It closely matches the provided design and adapts to desktop, tablet, and mobile screens.

## Features
- **Pixel-perfect design** matching the provided reference images
- **Bootstrap grid system** for layout and responsiveness
- **Interactive membership cards** (click to expand/collapse details)
- **Floating WhatsApp action button** for instant support
- **Modern, clean, and accessible UI**

## Structure
- `index.html` — Main HTML file
- `css/styles.css` — Custom styles and responsive tweaks
- `js/scripts.js` — JavaScript for interactivity

## How to Run
1. Clone or download this repository.
2. Open `index.html` in your browser.
3. The page will adapt to your device size (desktop, tablet, mobile).

## Interactivity
- Click on **FOUNDATION MEMBERSHIP** or **ECONOMY MEMBERSHIP** to reveal/hide their descriptions.
- The floating WhatsApp button opens a chat (link can be customized).

## Technologies Used
- HTML5
- CSS3 (with custom media queries)
- Bootstrap 5
- Bootstrap Icons
- JavaScript (vanilla)

## Accessibility & Responsiveness
- All sections scale and align together on all screen sizes.
- Sufficient color contrast and readable font sizes.
